# Security

- Report security issues to **contact@meshvault.ai** (subject: `skills-starter security`).
- These skills are instruction files, not a hosted service. Review every skill before enabling it on systems with real accounts.
- Never put credentials, tokens, or personal data into skill files or example outputs.
- Approval gates are policy, not a sandbox. Host tooling still needs proper auth scopes and human review for consequential actions.
